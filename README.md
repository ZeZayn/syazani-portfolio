# 🚀 syazani-portfolio

Personal portfolio website for **Muhammad Syazani** — Backend Developer & Junior DevOps Engineer based in Cyberjaya, Malaysia.

🌐 **Live site:** [ZeZayn.github.io/syazani-portfolio](https://ZeZayn.github.io/syazani-portfolio)

---

## 📋 Sections

- **About** — Introduction and profile status
- **Skills** — Tech stack: Node.js, Docker, n8n, MySQL, and more
- **DevOps** — Infrastructure & deployment practices
- **Projects** — CertFlow, TrainFlow, AI Personal Assistant
- **Experience** — Work history and background
- **Contact** — Formspree-powered contact form

---

## 🛠️ Built With

- HTML, CSS, JavaScript (vanilla — no frameworks)
- Canvas API for animated particle background
- 3D CSS transforms for card hover effects
- [Formspree](https://formspree.io) for contact form
- [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) + [Inter](https://fonts.google.com/specimen/Inter) fonts
- Hosted on **GitHub Pages**
